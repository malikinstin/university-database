University Database Management System

Overview

This project implements a relational database management system for a university using Microsoft SQL Server. The database stores and manages information about students, instructors, courses, enrollments, and grades while demonstrating key database design principles and SQL operations.

Features

- Creation of relational database tables
- Primary and Foreign Key constraints
- Student, Instructor, Course, Enrollment, and Grade management
- Data insertion and manipulation
- Transaction management using COMMIT and ROLLBACK
- Table modification using ALTER TABLE
- Data retrieval using SQL queries
- Sorting, filtering, and pattern matching operations

Database Structure

Instructor

Stores instructor information including office location and academic rank.

Course

Stores course details and links each course to an instructor.

Student

Stores student personal and enrollment information.

Takes

Tracks student course registrations.

Grade

Stores grades earned by students in specific courses.

Technologies Used

- SQL
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

Concepts Demonstrated

- Relational Database Design
- Entity Relationships
- Data Integrity Constraints
- Transaction Processing
- Query Optimization Basics
- Database Administration Operations
